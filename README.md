
Aslıhan ve Metehan - Mini Story Game
-----------------------------------

Bu proje, **Aslıhan ve Metehan** isimli kısa bir hikâye/etkileşimli oyun içerir.
Özellikler:
- Jetpack Compose tabanlı tek Activity
- Kısa 4 sahneli dallanmalı hikâye (her sahnede 2 seçenek)
- Basit ses efektleri (doğrudan assets içinde)
- Splash ekran ve adaptive icon
- GitHub Actions workflow ile otomatik APK üretimi

Nasıl derlenir:
1. ZIP'i aç ve GitHub'a yeni bir repo oluştur.
2. Tüm dosyaları repo'ya gönder.
3. Actions sekmesinden "Run workflow" çalıştır.
4. Artifact içinden `app-release.apk` indirebilirsin.
